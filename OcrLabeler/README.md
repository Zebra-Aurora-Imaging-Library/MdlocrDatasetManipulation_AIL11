Date: 2025/11/25

AIL Version: AIL 11

Type: Tool



Description:

OcrLabeler.exe is an OCR labeling tool. The tool was developed for internal needs and is made available as-is to the AIL user community. The tool includes a user guide to assist with the labeling process. This version is compatible with AIL 11.



Main Features:



* Interactively define bounding boxes using both keyboard and mouse.
* Label letters, numbers, and special characters.
* Utilize the DL OCR module to expedite the labeling process.
* Load and export annotations in an AIL-compatible JSON format.
